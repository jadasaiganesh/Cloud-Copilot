Amazon RDS (Relational Database Service)

💡 What is RDS?

Amazon RDS is a managed service that makes it easy to set up, operate, and scale a relational database in the cloud. It supports engines like MySQL, PostgreSQL, MariaDB, Oracle, and SQL Server.

👨‍💻 Real-World Use Case

Imagine you're building an e-commerce platform. Instead of managing database software and infrastructure yourself, you use RDS for automated backups, performance tuning, and high availability — so you can focus on your business.

🔄 How It Works
  1. Choose a DB engine (e.g., MySQL).
  2. Select instance type and storage options.
  3. Enable Multi-AZ for high availability (optional).
  4. Launch and connect via endpoint.
  5. Monitor and scale automatically.
🌟 Key Features
  • Supports multiple SQL engines
  • Automated backups and patching
  • Multi-AZ deployments for high availability
  • Read replicas for performance scaling
  • Integrated with IAM and VPC for security
  • Pay-as-you-go pricing

💸 RDS Pricing Models
  • On-Demand: Pay hourly for usage with no upfront commitment.
  • Reserved: Lower costs with 1 or 3-year terms.
  • Free Tier: 750 hours/month of db.t2.micro instances for eligible users.
